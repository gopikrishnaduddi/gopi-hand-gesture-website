# 🖐️ Gopi Hand Gesture Website

An interactive web project that demonstrates hand gesture-based UI effects using modern frontend techniques.

## 🚀 Features

* ✨ Cursor-based interactive effects
* 🔥 Fire, smoke, and spark animations
* 🖱️ Dynamic gesture response
* 🎨 Smooth and visually engaging UI

## 📁 Project Structure

```
.
├── codingsharks.html
└── README.md
```

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ▶️ How to Run

1. Download or clone the repository
2. Open `codingsharks.html` in your browser
3. Move your cursor to see interactive effects
